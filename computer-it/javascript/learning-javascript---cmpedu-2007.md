## 关于作者 ##
Shelley Powers 在实践中应用 Web 技术并发表各种与 Web 技术有关的文章已经长达 13 年之久。她最近在 O'Reilly 出版的书涉及语义 Web、Ajax、JavaScript 和 Web 图形等多个领域。她还是狂热的业余摄影师，同时也是 Web 开发的狂热爱好者。
